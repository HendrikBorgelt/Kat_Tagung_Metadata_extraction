# How to Use the Search Widget

## Basic search

Type any word or phrase into the search box. Results update as you type.

The search runs across **title**, **author names**, and **affiliations**
simultaneously (fuzzy matching — minor typos are tolerated).

## Filtering by field

Use the radio buttons below the search box to restrict results:

| Filter | Searches in |
|---|---|
| **All** | Title + author names + affiliations |
| **Title** | Abstract title only |
| **Author** | Author names only |
| **Affiliation** | Institution names only |

## Reading an abstract

In the **Full Details** widget (Option 3), each result has a **Read abstract →**
button.  Clicking it opens the original PDF in an inline reader page showing:

- Full title and year
- Presenting author (highlighted)
- All authors with their affiliations
- The PDF embedded directly in the page

## Clearing the search

Click the **×** button on the right side of the search box, or select all and delete.

## Tips

- Searching `"Berlin"` under **Affiliation** finds all abstracts with a Berlin-based author.
- Searching a surname under **Author** is the fastest way to find a specific person's work.
- The search is case-insensitive.
